Step 3. Create User Account

Please provide some additional details to set up the user account. Information collected here will only be used in our On-line Transaction Processing System. For more information about our data collection policies, see the EXIM Privacy Policy.

All fields are required unless indicated.

User ID

User ID can only contain characters from the following four groups:

  • English uppercase letters (A through Z)
  • English lowercase letters (a through z)
  • Numerals (0 through 9)
  • Non-alphabetic characters: hyphen (-), period (.), and underscore (_)

Password

Password should conform to the following guidelines:

  • Must be at least 8 chracters long
  • Must contain at least 1 character from each of the following four character groups:
    • English uppercase letters (A through Z)
    • English lowercase letters (a through z)
    • Numerals (0 through 9)
    • Non-alphabetic characters
  • Must not contain spaces
  • Must not be the same as the User ID

Challenge Question

Select a challenge question and response. We will use this to verify your identity when you call the EXIM offices for your password.
